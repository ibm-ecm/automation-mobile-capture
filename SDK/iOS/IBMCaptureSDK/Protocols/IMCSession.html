IMCSession

@protocol IMCSession <IMCEntity>

Describes an IMCSession. A session is an object created on the app that will be linked with a session on the server. It will contain the data that needs to be uploaded.

  • sessionId

    The session ID. sessionIds are guaranteed to be unique, negative values will be used for sessions which do not have a server counterpart, therefore have no gloabal session id
