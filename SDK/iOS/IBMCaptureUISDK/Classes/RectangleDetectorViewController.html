RectangleDetectorViewController

open class RectangleDetectorViewController : UIViewController

A UIViewController that presents a full screen RectangleImageDetectorCameraView to detect a rectangle and crop it. You should use a specialized verson of this UIViewController if you wish to detect and extract data from a Passport, USDL and Document.

  • State

    Describes the state of the ContainerViewController. This UIViewController cycles between a capturing state and a reviewing state where the user (if needed) can accept/reject the image.
